## Installation Steps

### Flashing the Image (SD Card)

**Please make sure to choose the file ending with `.img`**
After downloading and extracting the image, use `dd` to flash the image to the microSD card.

```bash
unzip openwrt-spacemit-k1-sbc-debX-ext4-pack-sdcard.img	
sudo dd if=/path/to/openwrt-spacemit-k1-sbc-debX-ext4-pack-sdcard.img of=/dev/your-device bs=1M status=progress
```

### Logging into the System

Logging into the system via the serial port.

Default User: root

Just hit enter to get the terminal, no password needed.
