# OpenWrt 香蕉派 BPI-F3 测试报告

## 测试环境

### 系统信息

- 系统版本：v23.05.2
- 下载链接：https://archive.spacemit.com/openwrt/releases/bl-v2.0.y/targets/spacemit/DEVICE_debX/
- 参考安装文档：https://bianbu-linux.spacemit.com/openwrt-docs/

### 硬件信息

- 香蕉派 BPI-F3
- 电源适配器
- microSD 卡一张
- USB to UART 调试器一个

## 安装步骤

### 刷写镜像（sd 卡）


**请务必选择以 `.img` 结尾的压缩包**
下载并解压镜像后，使用 `dd` 将镜像写入 microSD 卡。

```bash
unzip openwrt-spacemit-k1-sbc-debX-ext4-pack-sdcard.img	
sudo dd if=/path/to/openwrt-spacemit-k1-sbc-debX-ext4-pack-sdcard.img of=/dev/your-device bs=1M status=progress
```

### 登录系统

通过串口登录系统。

默认用户名： `root`

没有设置密码，按回车键登入终端。

## 预期结果

系统正常启动，能够通过板载串口登录。

## 实际结果

系统正常启动，成功通过板载串口登录。
